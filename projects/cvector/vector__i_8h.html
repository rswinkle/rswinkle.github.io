Definition at line 69 of file vector_i.c.

void cvec_i_copy ( void *  dest,
void *  src 
)

Makes dest an identical copy of src.

The parameters are void so it can be used as the constructor when making a vector of vector_i's. Assumes dest (the structure) is already allocated (probably on the stack) and that capacity is 0 (ie the array doesn't need to be freed).

Definition at line 109 of file vector_i.c.

vector_i* cvec_i_heap ( size_t  size,
size_t  capacity 
)

Creates a new vector_i on the heap.

Vector size set to (size > 0) ? size : 0; Capacity to (capacity > vec->size || (vec->size && capacity == vec->size)) ? capacity : vec->size + VEC_I_START_SZ in other words capacity has to be at least 1 and >= to vec->size of course.
