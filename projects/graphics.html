These are some screenshots from some of my various OpenGL (3.3) graphics programs. The projects these screenshots are from include Deferred Shading (diffuse/color, normal, position buffers and of course the final result), Shadow Mapping, Global Illumination using Instant Radiosity. and a random hermite spline rollercoaster program I wrote. All of these programs have full 6 degrees of freedom and controls like the game Descent(at least how I play it). I use SFML for my windowing/user input system as well as it's image loading routine for loading my textures. It's a cross platform library and I develop on linux so all of these screenshots are running on linux. I have compiled a couple of them in VS on windows and they work. Another thing to note is all of these use modern OpenGL 3.3 (equivalent to Direct3D 10.1 in capability). This means they all use GLSL vertex and fragment shaders. Eventually I will post some projects that use the geometry shader. They run on my 3 year old laptop (32 bit linux/windows with Nvidia 8600M GT) and my new desktop (64 bit linux/windows with Nvidia 430). I plan to eventually add more detailed individual descriptions, upload screenshots from a bunch more of my random projects and get the source code (at least for the common code) up on github.
